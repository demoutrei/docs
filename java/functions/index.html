Java Functions

Functions in programming are blocks of reuseable code. Utilizing functions reduces the construct of repetitive code.

In Java, functions are referred to as methods, of which functions that are bound to a class or object. Below is an example of a Java method:


public class Main {
  public static void main(String[] args) {
    System.out.println("Hello, World!"); // "Hello, World!\n"
  }
}

Methods are defined with the following keywords:

  1. public / private -- Whether the method is accessible from outside the class or solely exclusive from within the class only.
  2. static -- (Optional) Whether the method is a static method.
  3. (Required) Return type of the method.
  4. Unique identifier name of the method.
  5. (Optional) Method parameters.

The code of the functions are only executed when the function is called. To do so, the callable (function/method) is retrieved and suffixed with parentheses () and its corresponding arguments, if applicable. 


class Main {
  static void greet(String name) {
    System.out.printf("Hello, %s!%n", name);
  }

  public static void main(String[] args) {
    greet("demoutrei"); // "Hello, demoutrei!\n"
  }
}

Static Functions

Static functions are methods that don't require an instance object of its parent class. These types of methods are defined with a static keyword.


public class Main {
  public static void helloWorld() {
    System.out.println("Hello, World!");
  }
  
  public static void main(String[] args) {
    Main.helloWorld(); // "Hello, World!\n"
  }
}

Non-Static Functions

Non-static functions, also called "instance methods," are bound to a distinct instance of a class. Non-static functions are defined by opting out the static keyword in the function definition. 


class Player {
  String name;

  Player(String name) {
    this.name = name;
  }

  void greet() {
    System.out.printf("Hello, %s!%n", this.name);
  }
}


class Main {
  public static void main(String[] args) {
    Player player = new Player("demoutrei");
    player.greet(); // "Hello, demoutrei!\n"
  }
}

Function Parameters

Parameters are arguments that you can pass into a function/method. These are defined inside the parentheses () in the function definition line (next to the function name).

To create a parameter, define its data type, followed by the unique identifier name for the parameter: 


class Main {
  static void greet(String name) {
    System.out.printf("Hello, %s!%n", name);
  }
}

When defining multiple parameters, each are separated by commas.

Arguments

Arguments are variables/values that you pass into a matching parameter in the called function.

Arguments passed into the function call must match the amount of parameters there are, though not necessarily have to match the parameter name (although recommended).

In Java, arguments must be passed as positional, matching the expected parameter. 


class Main {
  static void introduce(String name, int age) {
    System.out.printf("Hello, my name is %s!%n", name);
    System.out.printf("I am %d years old%n", age);
  }

  public static void main(String[] args) {
    introduce("demoutrei", 20);
    /*
      "Hello, my name is demoutrei!\n"
      "I am 20 years old\n"
    */
  }
}

When passing multiple arguments, each must be separated by commas.

Function Returns

The return keyword is used to return a value from a executing function. If a variables/value is not present, it defaults to null .

The returned value must match the return type in the function definition. 


class Point {
  int x, y;

  Point() {
    this(0, 0);
  }

  Point(int x, int y) {
    this.x = x;
    this.y = y;
  }

  Point center() {
    return new Point(x / 2, y / 2);
  }
}

class Main {
  static int add(int x, int y) {
    return x + y;
  }
  
  public static void main(String[] args) {
    Point point = new Point(4, 6);
    Point center = point.center();
    System.out.printf("center = Point(%d, %d)%n", center.x, center.y); // "center = Point(2, 3)\n"
    System.out.printf("%d%n", add(center.x, center.y)); // "5\n"
  }
}
